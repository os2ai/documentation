# Release 0.8.10

Releasen indeholder:

- Opdatering af OpenWebUI 0.8.10, man kan læse mere om hvad den indeholder her:
[https://github.com/open-webui/open-webui/releases/tag/v0.8.10](https://github.com/open-webui/open-webui/releases/tag/v0.8.10)
- Support for proper tool call with Mistral model
Mulighed for at anvende tools (f.eks. til agentisk søgning) når man anvender Mistral modellen.
- Pinned vllm version
Fast vLLM version, sikkerhedstiltag for at sikre mod angreb.
- Ny RAG template
Så der anvendes forskellige templates til websøgning og til søgning i viden.
Templaten kan opdateres i brugergrænsefladen, men vil blive overskrevet ved næste release.
