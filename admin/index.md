# Admin

Her finder du informationer om hvordan du kan konfigurere din OpenWebUI som administrator og hvordan AAK tester nye releases.

Hver opmærksom på at noget af konfiguration kommer som "configuration as code" mens andet gemmes i databasen. Ændringer du laver i konfiguration kan derfor blive overskrevet når der kommer opdateringer. Tag gerne kontakt hvis du oplever ændringer (som du gerne vil beholde) bliver overskrevet ved opdateringer/genstart af applikationen.
